# Directory structure

This package is organised as:

- stage first (`as/`, `a2/`)
- learning level second
- topic third

Levels:

- level-1-core
- level-2-guided-cloze
- level-3-multi-round-cloze
- level-4-full-reconstruction

This matches a memorisation-first progression:

1. exact AO1 dictation
2. guided explanation cloze
3. multi-round cloze
4. full-sentence reconstruction
