A2 physical / electrochemistry / equilibria equations

Standard hydrogen electrode

  • Half-equation. \( \ce{2H+(aq) + 2e- <=> H2(g)} \)

Named electrode / Nernst examples in the syllabus

  • Copper(II) / copper. \( \ce{Cu2+(aq) + 2e- <=> Cu(s)} \)
  • Iron(III) / iron(II). \( \ce{Fe3+(aq) + e- <=> Fe2+(aq)} \)

Conjugate-acid / weak-acid / buffer forms

  • Weak acid. \( \ce{HA(aq) + H2O(l) <=> H3O+(aq) + A-(aq)} \)
  • Ethanoic acid. \( \ce{CH3CO2H(aq) + H2O(l) <=> H3O+(aq) + CH3CO2-(aq)} \)
  • Buffer removes added acid. \( \ce{CH3CO2-(aq) + H+(aq) -> CH3CO2H(aq)} \)
  • Buffer removes added alkali. \( \ce{CH3CO2H(aq) + OH-(aq) -> CH3CO2-(aq) + H2O(l)} \)
  • Blood buffer style. \( \ce{HCO3-(aq) + H+(aq) -> H2CO3(aq)} \)
  • Carbonic acid decomposition. \( \ce{H2CO3(aq) -> CO2(g) + H2O(l)} \)
  • Bicarbonate with alkali. \( \ce{HCO3-(aq) + OH-(aq) -> CO3^2-(aq) + H2O(l)} \)

Electrochemistry / transition-metal redox systems explicitly named

  • Permanganate + ethanedioate. \( \ce{2MnO4-(aq) + 5C2O4^2-(aq) + 16H+(aq) -> 2Mn2+(aq) + 10CO2(g) + 8H2O(l)} \)
  • Permanganate + iron(II). \( \ce{MnO4-(aq) + 5Fe2+(aq) + 8H+(aq) -> Mn2+(aq) + 5Fe3+(aq) + 4H2O(l)} \)
  • Copper(II) + iodide. \( \ce{2Cu2+(aq) + 4I-(aq) -> 2CuI(s) + I2(aq)} \)

Catalysis example used in kinetics

  • Overall uncatalysed reaction. \( \ce{S2O8^2-(aq) + 2I-(aq) -> I2(aq) + 2SO4^2-(aq)} \)
  • Catalytic step 1. \( \ce{S2O8^2-(aq) + 2Fe2+(aq) -> 2SO4^2-(aq) + 2Fe3+(aq)} \)
  • Catalytic step 2. \( \ce{2Fe3+(aq) + 2I-(aq) -> 2Fe2+(aq) + I2(aq)} \)

A2 inorganic chemistry

28 Transition elements

Copper(II) aqua complex and ligand exchange

  • Hydrated ion. \( \ce{Cu2+(aq) + 6H2O(l) <=> [Cu(H2O)6]2+(aq)} \)
  • With NH3, limited. \( \ce{[Cu(H2O)6]2+(aq) + 2NH3(aq) <=> Cu(OH)2(s) + 2NH4+(aq) + 4H2O(l)} \)
  • With excess NH3. \( \ce{[Cu(H2O)6]2+(aq) + 4NH3(aq) <=> [Cu(NH3)4(H2O)2]2+(aq) + 4H2O(l)} \)
  • With OH-. \( \ce{[Cu(H2O)6]2+(aq) + 2OH-(aq) <=> Cu(OH)2(s) + 6H2O(l)} \)
  • With chloride. \( \ce{[Cu(H2O)6]2+(aq) + 4Cl-(aq) <=> [CuCl4]2-(aq) + 6H2O(l)} \)

Cobalt(II) aqua complex and ligand exchange

  • Hydrated ion. \( \ce{Co2+(aq) + 6H2O(l) <=> [Co(H2O)6]2+(aq)} \)
  • With NH3, limited. \( \ce{[Co(H2O)6]2+(aq) + 2NH3(aq) <=> Co(OH)2(s) + 2NH4+(aq) + 4H2O(l)} \)
  • With excess NH3. \( \ce{[Co(H2O)6]2+(aq) + 6NH3(aq) <=> [Co(NH3)6]2+(aq) + 6H2O(l)} \)
  • With OH-. \( \ce{[Co(H2O)6]2+(aq) + 2OH-(aq) <=> Co(OH)2(s) + 6H2O(l)} \)
  • With chloride. \( \ce{[Co(H2O)6]2+(aq) + 4Cl-(aq) <=> [CoCl4]2-(aq) + 6H2O(l)} \)
