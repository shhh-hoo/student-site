A2 aromatic chemistry, halogen compounds, phenol, and related hydroxy-compound reactions.

A2 organic chemistry

30 Arenes

Electrophilic substitution of benzene

  • Chlorination. \( \ce{C6H6 + Cl2 -> C6H5Cl + HCl} \)
  • Bromination. \( \ce{C6H6 + Br2 -> C6H5Br + HBr} \)
  • Nitration. \( \ce{C6H6 + HNO3 -> C6H5NO2 + H2O} \)
  • Friedel–Crafts alkylation. \( \ce{C6H6 + CH3Cl -> C6H5CH3 + HCl} \)
  • Friedel–Crafts acylation. \( \ce{C6H6 + CH3COCl -> C6H5COCH3 + HCl} \)

Note: Use AlCl3 / AlBr3 catalysts for halogenation and Friedel–Crafts; concentrated H2SO4 for nitration.

Electrophile generation equations often written with the mechanisms

  • Bromination support. \( \ce{Br2 + AlBr3 -> Br+ + AlBr4-} \)
  • Nitration support. \( \ce{HNO3 + H2SO4 -> NO2+ + HSO4- + H2O} \)

Side-chain oxidation and ring hydrogenation

  • Methylbenzene oxidation. \( \ce{C6H5CH3 + 3[O] -> C6H5CO2H + H2O} \)
  • Benzene hydrogenation. \( \ce{C6H6 + 3H2 -> C6H12} \)

Methylbenzene halogenation pathways

  • Ring substitution (catalyst). \( \ce{C6H5CH3 + Cl2 -> CH3C6H4Cl + HCl} \)
  • Side-chain substitution (UV). \( \ce{C6H5CH3 + Cl2 -> C6H5CH2Cl + HCl} \)

Note: CH3C6H4Cl represents the ortho / para isomer mixture from ring substitution.

31 Halogen compounds

Halogenoarenes from arenes

  • Benzene -> chlorobenzene. \( \ce{C6H6 + Cl2 -> C6H5Cl + HCl} \)
  • Methylbenzene -> chloromethylbenzene isomers. \( \ce{C6H5CH3 + Cl2 -> CH3C6H4Cl + HCl} \)

32 Hydroxy compounds

Alcohol + acyl chloride

  • Ethyl ethanoate example. \( \ce{CH3COCl + C2H5OH -> CH3CO2C2H5 + HCl} \)

Phenol preparation from phenylamine

  • Diazotisation. \( \ce{C6H5NH2 + NaNO2 + 2HCl -> C6H5N2+Cl- + NaCl + 2H2O} \)
  • Hydrolysis of diazonium salt. \( \ce{C6H5N2+Cl- + H2O -> C6H5OH + N2 + HCl} \)

Phenol reactions

  • With NaOH. \( \ce{C6H5OH + NaOH -> C6H5ONa + H2O} \)
  • With sodium. \( \ce{2C6H5OH + 2Na -> 2C6H5ONa + H2} \)
  • With bromine water. \( \ce{C6H5OH + 3Br2(aq) -> C6H2Br3OH(s) + 3HBr} \)
  • With dilute nitric acid. \( \ce{C6H5OH + HNO3 -> C6H4(OH)NO2 + H2O} \)

Note: Nitration gives a mixture of 2-nitrophenol and 4-nitrophenol.

Azo coupling with phenol

  • Representative equation. \( \ce{C6H5N2+Cl- + C6H5OH + NaOH -> HOC6H4N=NC6H5 + NaCl + H2O} \)
